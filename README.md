# コタれん

## コタれんって

* 押尾コータローさんの練習曲の進捗を管理できます。
* チューニングで曲の検索を行えます。
* 練習の参考になる YouTube の関連動画を自動検索できます。
* 他のユーザの進捗情報を閲覧できます。
